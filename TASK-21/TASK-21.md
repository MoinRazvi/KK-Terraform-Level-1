# Terraform Level-1 Task-21: Create CloudWatch Log Group and Log Stream

## 📌 Task Objective

Create an AWS CloudWatch Log Group and Log Stream using Terraform.

### Requirements

* Log Group Name: `datacenter-log-group`
* Log Stream Name: `datacenter-log-stream`
* Region: `us-east-1`

Terraform working directory:

```bash
/home/bob/terraform
```

---

## Step 1: Navigate to Working Directory

```bash
cd /home/bob/terraform
```

---

## Step 2: Create `main.tf`

```hcl
provider "aws" {
  region = "us-east-1"
}

resource "aws_cloudwatch_log_group" "datacenter_log_group" {
  name = "datacenter-log-group"
}

resource "aws_cloudwatch_log_stream" "datacenter_log_stream" {
  name           = "datacenter-log-stream"
  log_group_name = aws_cloudwatch_log_group.datacenter_log_group.name
}
```

---

## Step 3: Initialize Terraform

```bash
terraform init
```

---

## Step 4: Validate Configuration

```bash
terraform validate
```

Expected output:

```bash
Success! The configuration is valid.
```

---

## Step 5: Apply Configuration

```bash
terraform apply -auto-approve
```

---

## Step 6: Verify Log Group Creation

```bash
aws logs describe-log-groups \
--log-group-name-prefix datacenter-log-group \
--region us-east-1
```

Verify:

```text
datacenter-log-group
```

---

## Step 7: Verify Log Stream Creation

```bash
aws logs describe-log-streams \
--log-group-name datacenter-log-group \
--region us-east-1
```

Verify:

```text
datacenter-log-stream
```

---

## 🔍 Key Concepts Learned

### Amazon CloudWatch Logs

CloudWatch Logs is used to:

* Store application logs
* Monitor system events
* Troubleshoot issues
* Analyze operational data

---

### Log Group

A Log Group acts as a container for multiple log streams.

```hcl
resource "aws_cloudwatch_log_group"
```

Creates a CloudWatch Log Group.

---

### Log Stream

A Log Stream contains a sequence of log events generated by a specific source.

```hcl
resource "aws_cloudwatch_log_stream"
```

Creates a CloudWatch Log Stream.

---

### Resource Dependency

```hcl
log_group_name = aws_cloudwatch_log_group.datacenter_log_group.name
```

Terraform automatically creates the log group before creating the log stream.
